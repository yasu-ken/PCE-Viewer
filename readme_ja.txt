
PC Engine (Turbo Graphics 16) Viewer

・ロム (くれぐれも自分が所有しているカードのものにしてください)
・YAME セーブデータ
・MagicEngine セーブデータ
・ISO (ご自分が所有しているCDから、第二トラックのバイナリデータを取り出したもの)

などを読み込んで、無圧縮の画像部分があればそれを表示します。

;------------------------------------------------------------
使い方

exeを起動しますと、読み込みダイアログが開くので
データファイルを読み込んでください。

Windows7 32bit 64bitで動作確認しました。

;------------------------------------------------------------
ビルド方法
vc++6.0でビルドします。
DirectXのバージョンは9.0cくらい・・・いまどき手に入るか、システムにインストールできるのかは不明です

Project.dswを開き、
Build -> Batch Build ->「Project -Win32 el」にチェックが入ってるのを確認
「Rebuild All」でビルドします。


デバッグモードでビルドしてもなぜか動きません。理由は忘れました・・・
